# OpendoorBackendChallenge

### Getting Started

##### Running local:

..* Clone GitHub repo.
..*  In terminal: ```npm install```
..* ```npm start```
..* Open http://localhost:3000 in a browser window.
..* Submit queries to /listings endpoint.

##### On Heroku:

..* Open https://query-real-estate-listings.herokuapp.com/ in a browser window.
..* Submit queries to /listings endpoint.
